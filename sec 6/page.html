Introduction to HTML

Terms

Web Pages

Documents that are written in a language called HTML

HTML

Stands for HyperText Markup Language

HTML tags

Special codes that tell the web browser how to display the HTML document

Web Browser

An application that can interpret HTML and display the document in the format according to markup tags. Example: Firefox, Google Chrome.
